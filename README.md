# geocoder-microservice

## Dependencies
* Ruby 2.6.3
* Bundler 2.1.4
* Sinatra 2.0.0
* Puma 4.3.0

## Installation
```
git clone git@github.com:natalya-bogdanova/auth_microservice.git && cd auth_microservice
bundle install
```

## Run application:
```
bin/puma
```

## Run specs and console
```
bin/console
bin/rspec
```
