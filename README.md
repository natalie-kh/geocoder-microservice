# geocoder-microservice

## Dependencies
* Ruby 2.6.3
* Bundler 2.1.4
* Sinatra 2.0.0
* Puma 4.3.0

## Installation
```
git clone https://github.com/natalya-bogdanova/geocoder-microservice.git && cd geocoder-microservice
bundle install
```

## Run application:
```
bin/app
```

## Run specs and console
```
bin/console
bin/rspec
```
